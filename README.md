# blog

> based on hexo

## 安装

```bash
npm i hexo-cli -g
```

## 安装主题
在终端窗口下，定位到 Hexo 站点目录下。使用 Git checkout 代码：

```bash
git clone https://github.com/theme-next/hexo-theme-next themes/next
```

## 本地启动

```bash
npm start
```

## 发布blog

```bash
npm run pub
```

## 新增日志

```bash
hexo new post 'new post'
```
