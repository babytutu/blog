## root

```bash
$ sudo -s
```
## list file

```bash
$ cd
$ cd ..
$ ls
$ ls -l
```

## make/replace file

```bash
$ mkdir xxx
$ rm -rf xxx
```
## copy/move file

```bash
$ cp -r pathA pathB
$ mv pathA path B
```
## change owner/permission

```bash
$ sudo chown -R user path/file
$ sudo chmod -R 777 path/file
```
